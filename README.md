# ЛАЙМ — лендинг (CRM-внедрение × вайбкодинг)

Одностраничный лендинг студии **ЛАЙМ**: внедрение amoCRM и Битрикс24, интеграции, ИИ-агенты, кастомная разработка на вайбкодинге, свои SaaS и MVP.

Визуальный язык — в стиле Hut 8 (Awwwards): светлая «бумажная» палитра, тонкие линии, премиальная типографика и **анимированный изометрический «живой город» на фоне**, который пустой в начале и достраивается в единый механизм по мере скролла.

## Запуск локально

Это статический сайт без сборки. Любой статический сервер подойдёт:

```bash
python3 -m http.server 8077
# открыть http://localhost:8077
```

## Структура

| Файл | Назначение |
| --- | --- |
| `index.html` | Весь сайт: разметка + инлайновый движок «живого города» (2D canvas, ванильный JS). Самодостаточный. |
| `image-slot.js` | Кастомный элемент `<image-slot>` — плейсхолдеры под скриншоты кейсов и hero-фото. |
| `design_handoff_lime_landing/` | Исходный дизайн-референс (прототип `ЛАЙМ.dc.html`, спецификация в `README.md`). |

`index.html` собран из прототипа `design_handoff_lime_landing/ЛАЙМ.dc.html`: контейнеры `<helmet>`/`<x-dc>` дизайн-композера развёрнуты в обычные `<head>`/`<body>`, движок переведён в исполняемый `<script>` с дефолтными параметрами (`accent #B8E020`, `cityDensity Средний`, `liveBackground true`).
